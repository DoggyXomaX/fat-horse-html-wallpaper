# Fat horse DVD Wallpaper

[Plasma HTML Wallpaper](https://store.kde.org/p/1324580) adaptation of the "walking fat horse.gif" with mimics to the iconic DVD screensaver.

![](.assets/preview.gif)

## Install

* Clone the repository
* In the HTML Wallpaper settings, set the path to **index.html** with the `file:///` prefix
* And OK

![](.assets/html-wallpaper-settings.webp)
